Равенство медиан¶

Непрерывное распределение¶

Проверим гипотезу о равенстве медиан для распределения роста в трех сегментах. Мы не можем применить критерий Манна-Уитни, так как распределения выборок не однородны, воспользуемся критерием Уилкоксона, при этом мы должны понимать, что выборки зависимые (в разных сегментах имеются одни и те же игроки) и разного размера. Будем брать подвыборки и считать среднюю ошибку первого рода.

from tqdm import tqdm
def wilc(data1, data2):
    n = min(data1.shape[0], data2.shape[0])
    c = 0
    for i in range(1000):
        c += st.wilcoxon(
            data1.sample(n, replace=False),
            data2.sample(n, replace=False)
            ).pvalue < 0.05

    return c / 1000

Бутстрап¶

Перед нами стоит задача: проверить результаты гипотез об однородности распределений с помощью бутстрапа.

def bootstrap_confidence_interval(sample, stat, alpha=0.05):
    """
    Стоит эфронов 1 - alpha доверительный интервал для выбранной статистики stat
    """
    n = sample.shape[0] // 2
    distribution_lst = []
    for _ in range(100):
        a_ = np.random.choice(sample, size=n, replace=True)
        distribution_lst.append(stat(a_))

    return np.percentile(distribution_lst, q=[alpha / 2 * 100, (1 - alpha / 2) * 100])

def bootstrap_kolmogorov(data1, data2):
    """
    Возвращает среднее значение pvalue
    основанное на проверки однородсти двух выборок критерием Колмогорова
    """
    n = min(data1.shape[0], data2.shape[0])
    pvalues = []
    for _ in range(100):
        pv = st.ks_2samp(data1.sample(n).values,
                         data2.sample(n).values
                         ).pvalue
        pvalues.append(pv)

    return sum(pvalues) / len(pvalues)

def bootstrap_confidence_interval_2_samples(data1, data2, alpha=0.05, k=1):
    """
    Стоит эфронов 1 - alpha доверительный интервал для моментов порядка k
    """
    n = min(data1.shape[0] // 2, data2.shape[0] // 2)
    distribution_lst = []
    for _ in range(100):
        a_ = (data1.sample(n).values - data2.sample(n).values)
        distribution_lst.append(moment(a_, k))

    return np.percentile(distribution_lst, q=[alpha / 2 * 100, (1 - alpha / 2) * 100])

Мнение¶

По моему мнению бутстрап довольно мощный способ проверки статистических гипотез относительно каких либо параметров распределения. Однако не совсем понятно как его можно использовать для проверки гипотез об однородности выборок. Также бустрап очень хорош, когда распределение выборок не подходит для статистических тестов. На моих данных бутстрапом пришлось воспользоваться только для подтверждения полученных выводов о равенствах медиан, но если бы данные не подходили под требования критериев, то гораздо более адекватными методом проверки гипотез оказался бы бутстрап.

Для проверки гипотез о распределении выборки или однородности двух выборок я точно буду пользоваться статистическими критериями, а не эвристиками для проверки этих гипотез.
